Fusebox 5.1 Core Files

The simplest way to install Fusebox 5.1 is to copy this (fusebox5)
directory to your webroot.

Alternatively, you can create a mapping /fusebox5 that points to
this directory.

Then install the Fusebox 5.1 Skeleton application template
(see the Downloads section of fusebox.org) and test that
it works.

Fusebox 5.1 supports:
- ColdFusion MX 6.1 and higher
- BlueDragon 7.0 Beta and higher
- Railo 1.0.0.025 and higher

For more details, see the Fusebox 5 documentation:

	http://fuseboxframework.org/index.cfm?fuseaction=documentation.WhatsNewInFusebox5
